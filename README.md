# Taxi-Analysis-Dashboard
Power BI dashboard analyzing taxi bookings, revenue, cancellations, and customer insights using real-world data.

🚖 1 in 4 Taxi Bookings Get Cancelled — Why?

I analyzed a taxi booking dataset using Power BI to understand customer behavior, revenue patterns, and operational inefficiencies.

⚠️ Dataset used is for learning purposes.

🔧 Tools Used
Excel | Power Query | Power BI | DAX

📊 Key Insights
• Nearly 25% of bookings are cancelled — a major revenue leakage. This indicates issues in pricing, driver availability, or customer reliability.

• UPI dominates as the payment method, showing strong digital adoption — businesses should optimize cashback/offers here.

• Morning hours generate the highest revenue, making it the peak demand window — surge pricing strategies can be applied.

• Auto category gives the highest revenue per KM, suggesting better cost-efficiency compared to other vehicle types.

• Customer retention is only ~30%, which signals poor user experience or lack of loyalty programs.

💡 What This Means
This analysis shows that improving cancellation rates and retention can significantly increase revenue without acquiring new customers.

📈 Key Learning
I realized data analysis is not about dashboards — it's about identifying problems and suggesting actionable solutions.
